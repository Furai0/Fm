для начала необходимо получить рут права
-
По умолчание пользователь adisadmin их имеет, а adis - нет

Для этого заходим под adisadmin, пароль обычно Qwerty123321 или 1 (СКРИНЫ1, 2)

Чтобы дать права adis вводим команду в терминале fly (СКРИН3, 4)

sudo usermod -aG sudo adis
Жмем enter и перезагружаемся
--
Затем выделяем все объекты мышью или сочитанием клашив ctrl+A в папке install (СКРИН5), нажимаем по какому-либо объекту в папке правой кнопкой мыши и жмем СВОЙСТВА (СКРИН6) и в графе дискреционные атрибуты проставляем все галочки на чтение, запись и исполнение (не трогаем специальные флаги) (СКРИН7)

Запускаем установочный скрипт INSTALL.sh (СКРИН8), ждем конца установки и закрываем окно.
